HTML adalah dasar dari semua website. Dengan HTML kita bisa membuat struktur halaman web.

Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa untuk membuat struktur website.

Contoh HTML

<html>
  <body>
    <h1>Hello World</h1>
  </body>
</html>

Dengan belajar HTML, kamu sudah memulai perjalanan menjadi web developer.
